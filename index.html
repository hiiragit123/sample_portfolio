バブルソート（Bubble Sort）

初級

学習日: 2024年3月15日

プロジェクトの概要

配列の要素を順番に比較して、隣接する要素の大小関係が正しくない場合は交換する単純なソートアルゴリズムを学習しました。

実装したコード

def bubble_sort(arr): n = len(arr) for i in range(n): for j in range(0, n-i-1): if arr[j] > arr[j+1]: arr[j], arr[j+1] = arr[j+1], arr[j] return arr # テスト numbers = [64, 34, 25, 12, 22, 11, 90] sorted_numbers = bubble_sort(numbers.copy()) print(f"元の配列: {numbers}") print(f"ソート後: {sorted_numbers}")

つまずいたポイントと解決方法

課題1: 配列の要素交換

Python での配列要素の交換方法がわからず、一時的に苦戦しました。

解決策: Pythonのマルチ代入（a, b = b, a）を使うことで簡単に解決できることを学びました。

課題2: ループの範囲設定

外側のループと内側のループの範囲設定が難しかったです。

解決策: 紙に書いて動きを確認することで理解できました。

学んだこと

  • 配列の要素を交換する方法
  • 二重ループの使い方
  • アルゴリズムの時間計算量（O(n²)）の概念
  • デバッグの重要性

線形探索（Linear Search）

初級

学習日: 2024年3月20日

プロジェクトの概要

配列の先頭から順に探索していく基本的な探索アルゴリズムを学習しました。

実装したコード

def linear_search(arr, target): for i in range(len(arr)): if arr[i] == target: return i return -1 # テスト numbers = [4, 2, 7, 1, 9, 5] result = linear_search(numbers, 7) print(f"7の位置: {result}")

実行結果のスクリーンショット

学んだこと

  • シンプルだが効果的な探索方法
  • 最悪の場合の時間計算量（O(n)）
  • 配列の操作方法
